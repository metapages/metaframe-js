## framejs.io: The ultimate user controlled web component

Generate and run self contained JavaScript iframes (metaframes) using AI — from chat interfaces or directly from the Command Line Interface, or via APIs.

The pages are embeddable, editable, customizable chunks of javascript code. Standalone, or as input / output pipes. They are the ideal composable unit for dashboards, editable or highly user customizable websites, scientific notebook components, visualization snapshots, and complete mini websites. They are public addressable, when wanted: immutable, and open graph accessible. The give the user control and increased safety: blast radius limited, and no hidden state rather complete visibility.
